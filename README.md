## Huafu Convenience Store Information Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/370e93e383ed4c7681cebb97c7a98526/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Change Password
3. Personal Information
4. Supplier Information Management
5. Product Management
6. Cashier Management
7. Cashier List Management


**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. Basic Data Management (supplier type management, regional type management, supplier credit level type management, product type management)
4. Supplier Information Management
5. Product Management
6. Purchase Management
7. Cashier Management, Cashier List Management
8. Employee Management
